|   | 命令 | 描述 | 其他 |
|---|------|------|------|
|   | C-x p f (project-find-file)     |   查找文件   |     |

### 2.1 查找文件

（1） 输入命令 `C-x-p-f` , 

（2）输入文件的部分名称，

（3）`Tab`，会显示当前目录下符合条件的文件列表

（4）选择具体的文件，`enter` 进入文件
