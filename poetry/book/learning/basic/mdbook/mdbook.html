一、概览

mdBook 是一个开源命令行工具, 由 rust官方维护, 可以简单快捷的创建markdown格式的电子书

rust语言的官方教程就是由 mdBook构建的

主要具有下列优点:

  • Markdown 格式可以让书写更简洁

  • 集成了简单的搜索功能

  • 代码高亮

  • 自定义一些主题

  • Preprocessors 各种预处理器可以预先处理文章内容

  • Backends 各种后端模块，可以渲染文章

  • 原生支持rust的代码测试

二、基本使用

2.1 安装


	cargo install mdbook

2.2 创建一本书


	mdbook init crafting-interpreters-zh
